# csv-to-json-har

### Prerequisites
To run this project, you need to following installed on your machine:
- Python 3
- Tkinter

### Running the Project
Open the terminal and navigate to the src folder. You can then run the project with the following command:

```
python main.py
```

Running the project with no command line arguments should open the GUI. Try adding some command line arguments to open
the CLI:

```
python main.py arg1 arg2 arg3
```
