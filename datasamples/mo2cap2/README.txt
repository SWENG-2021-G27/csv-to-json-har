########################
Downloading the Code and Datasets
########################
Use the script download.sh to download the code and datasets. You would need to read and review the 
configuration under conf.ig before you can proceed with downloading the code and datasets.
The full package includes:
- training data
- test sequences with ground truth
- Matlab code for evaluation on the test data
